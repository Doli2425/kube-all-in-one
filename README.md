# kube-all-in-one

Kubernetes 기초 실습을 모아놓은 개인 학습용 저장소입니다.  
Pod, Deployment, Service, ConfigMap, Ingress 등  
클러스터 기본 기능을 단계별로 실습하며 정리했습니다.

## 📂 구성
- `03` : kubectl 기본 명령어 실습
- `04` : Pod / ReplicaSet 기초 실습
- `05` : Deployment, DaemonSet 실습
- `06` : Service(ClusterIP/NodePort/LoadBalancer) 실습
- `07` : ConfigMap / Secret 실습
- `08` : Volume / PVC / PV 실습
- `09` : Ingress 실습
- `10~19` : 다양한 오브젝트 조합 및 실습 문제 풀이
- `labs` : 종합 실습 및 테스트용 매니페스트

## 🎯 목적
- Kubernetes 오브젝트 동작 원리 이해
- 실전 환경 대비 명령어 & YAML 실습
- Pod → Service → Deployment → Ingress 흐름 익히기
- 클러스터 실습 파일 체계적으로 정리

> 이 저장소는 교육 및 개인 학습용이며 예제 중심으로 구성되어 있습니다.
